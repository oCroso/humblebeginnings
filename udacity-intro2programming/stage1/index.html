HTML

HTML (HyperText Markup Language) is the structure from which pages are built. You can use it to lay out the box strucutre that contains the different elements of your page. Web pages like Wikipedia are great examples of pages with deep layers of html strucutre. You use div to divide up the page into multiple blocks that you use for the structure! Here's a link to W3C's page on html for more info: w3schools HTML tutorials


CSS

CSS (Cascading Style Sheets) are used to apply the "decoration" or design to your webpage. You can think of HTML like the Home, and CSS like the interior decorator. If I want to change the color of the background, create shapes, define the size of elements, etc, I can use my CSS to do so. A great example of how you can use CSS is:

h1 {
    background-color : black;
    color: white;
}



Here we see that all elements labeled h1 will now have a black background with white text. But you can also specify a class using class tags. For example:

.yourclassname {
    background-color : black;
    color: white;
}



Now, anything referenced by your html as belonging to the class .yourclassname will have the settings that h1 previously had. If you'd like to see the CSS "bible" click HERE!
