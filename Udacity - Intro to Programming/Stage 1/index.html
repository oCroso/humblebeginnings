CSS

CSS (Cascading Style Sheets) are used to apply the "decoration" or design to your webpage.
You can think of HTML like the Home, and CSS like the interior decorator.
If I want to change the color of the background, create shapes, define the size of elements, etc,
I can use my CSS to do so.
A great example of how you can us CSS is:

h1 {
   background-color : black;
   color: white;
}



Here we see that all elements labeled h1 will now have a black background with white text.
But you can also specify a class using class tags. For example:

.yourclassname {
   background-color : black;
   color: white;
}



Now, anything referenced by your html as belonging to the class .yourclassname will have
the settings that h1 previously had.
